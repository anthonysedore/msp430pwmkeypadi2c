# msp430pwmkeypadi2c
MSP430 PWM Output w/ Keypad Matrix & i2c LCD, used an MSP430FR2355 microcontroller to control 
two DC motors using an H-Bridge Controller, L298. H-Bridge Controller is connected to 12v battery.
